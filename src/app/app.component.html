This app imports a core module and a shared feature module. It also demonstrates Form Controls and Reactive Forms. The answer boxes automatically update when input values are changed.

Core module: this provides a set of Material Design buttons for reuse throughout the app.

Shared module: this provides a simple exported MathLib class with methods add(a, b) and mult(a, b).
